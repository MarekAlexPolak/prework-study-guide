HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element define semantic elements in a document that have thematic content.
  • The footer element is located at the bottom of the webpage and contains author, copyright, contact, sitemap, and navigation.

CSS

  • Margin property indicates space added around the outside of an element.
  • Padding property indicates space added around the inside of an element.
  • "*" indicates that all the elements in the document will have the code applied to them.
  • You can assign properties such as height, allignment, font size etc to the different html elements independently.

Git

  • Git status checks what branch we are currently on.
  • Git checkout -b branch-name: creates a new branch and switches to it.
  • Git checkout main: returns to the main branch.
  • Git push origin branch-name: pushes the current branch changes to github.
  • Git pull origin main: saves a copy of the main branch onto your system.
  • Git add -A: adds all modifications to the current branch to the staging area.
  • Git commit -m: saves changes to the branch you are currently working on

JavaScript

  • All variables are defined with var: var x = "string".
  • All java script code is read top to bottom so any variables or functions required for the base code must be above it.
  • You can use conditional if, else if, and else statements to control the flow of code.
  • You can filter through array indexes with loops.
  • for(i=0; i < desiredlength; x++){}: Syntax for a for loop in JavaScript.
  • // is used to comment out a single line of code.
  • /* Enter code here */ is used to comment out an entire section of code.
